# Getting started

## 1. Clone this repo

```
git clone https://github.com/sakisplus/Learn-React.git
```
## 2. Go to project dorectory and run the istall script
```
cd Learn-React
./install.sh
```

## 3. Start the engines
```
docker compose up -d
```

> To stop running modules use ``docker-compose down`` command
